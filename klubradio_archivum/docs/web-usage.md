# Klubrádió Archivum Web App Usage Guide

This document provides a guide for using the web version of the Klubrádió Archivum application. The web app is designed to provide a lightweight, streaming-only experience, allowing you to access Klubrádió's archived programs directly in your web browser.

---

## Key Features of the Web App

*   **Podcast Browsing:** Explore a wide range of archived podcasts.
*   **Episode Streaming:** Listen to any episode directly from your browser.
*   **Playback Controls:** Standard audio controls including play, pause, and seek.
*   **Search Functionality:** Easily find your favorite shows or episodes.

---

## Important Limitations (Web Version)

Due to the nature of web browsers and platform constraints, the following features available in the mobile application are **not supported** in the web version:

*   **No Downloads:** Episodes cannot be downloaded for offline listening. The web app requires an active internet connection for streaming.
*   **No Subscriptions:** You cannot subscribe to podcasts or manage auto-downloads.
*   **No Offline Playback:** All content is streamed live; there is no local storage for offline access.
*   **No Background Audio (Limited):** Audio playback may be interrupted if you switch tabs, close the browser, or put your device to sleep, depending on browser and operating system settings.

---

## How to Use

1.  **Access the App:** Navigate to `https://play.gi-td.com/` in your web browser.
2.  **Browse/Search:** Use the navigation tabs to browse podcasts or search for specific episodes.
3.  **Stream Episodes:** Click on any episode to start streaming its audio. Use the playback controls at the bottom of the screen.
